Небольшой проект для практики работы с данными. Взял CSV с продажами, почистил его на Python, загрузил в PostgreSQL и собрал дашборд в Redash.
 
Файлы:

- data/orders_raw.csv — исходные данные
- etl/ — скрипты обработки
- queries.sql — запросы для дашборда
